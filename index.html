Mon Anti-sèche des commandes du terminal

Répertoires

  1. pwd : Affiche le chemin des répertoires actuels
  2. cd (directory) : Changement de répertoire
  3. cd . . : Accédez au répertoire parent
  4. ls : Lister le contenu du répertoire
  5. ls -la : Lister le contenu caché du répertoire
  6. mkdir (directory) : Créer un nouveau répertoire nommé (directory)

Fichiers

  1. rm (files) : Supprime (files)
  2. rm -r (directory) : Supprime Répertoire (directory)
  3. rm -f : Forcer la suppression (ajouter r pour supprimer répertoire)
  4. mv : Renommer fichier
  5. cp : Copier fichier
  6. touch : Créer fichier

Rechercher

  1. find : Rechercher fichier
  2. grep : Rechercher un mot ou un motif dans un fichier

Réseau

  1. ping : ping host et affiche le statut
  2. curl -0 : Télécharge un fichier
  3. ssh : Etabli une connexion SSH
  4. scp : Copier fichier

Autorisations

  1. chmod : Change les permissions
  2. chown : Change les permissions des utilisateurs et des groupes

Processus

  1. ps ax : Sortie des processus en cours d'éxécution
  2. top : Afficher les informations sur les processus en cours d'éxécution
  3. kill : Quitter un processus avec ID
